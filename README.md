TASO Engine

Human-Centered Shogi Engine Wrapper

⸻

概要

TASO Engine は、既存の強力な将棋AI（dlshogi・やねうら王）を
人間が実戦で使いこなすことを目的として設計された
USI互換の将棋エンジン・ラッパーです。

本エンジンは「最善手を出すAI」ではありません。
勝っている局面では“人間が間違えにくい勝ち方”を、
負けている局面では“相手が崩れやすい道”を選ぶ
という思想に基づいて動作します。

将棋GUIからは通常のUSIエンジンとして扱えますが、
内部では以下の要素を統合的に制御しています。

⸻

主な特徴

1. BUNKER（勝勢の崖）判定
	•	評価値は優勢だが、最善手と次善手の差が極端に大きい局面を検出
	•	AI的には簡単だが、人間には危険な局面を可視化
	•	GUI上に「バンカー局面」として表示可能

⸻

2. HumanScore（人間勝率）の導入
	•	局面・評価値・選択肢の幅から
**「人間が勝ち切れる確率」**を推定
	•	HumanScore に応じて戦い方を自動切替
	•	🛡 受けて勝つ
	•	⚖ バランス
	•	🔥 攻めて勝つ

⸻

3. COMEBACK（逆転志向ロジック）
	•	劣勢時に「評価値を戻す手」ではなく
相手がミスしやすい展開を優先
	•	一撃必殺／事故誘発／持久戦 などに自動分岐

⸻

4. SUDDEN DEATH（頓死誘発モード）
	•	劣勢時、あえて最善手を外し
相手が崩れやすい筋を狙って選択
	•	MultiPV下位の手を戦略的に採用

⸻

5. 終盤エンジン切替
	•	必至・詰みが見えた段階で
dlshogi → やねうら王 に自動切替
	•	「勝ち切り」に特化した動作

⸻

想定用途
	•	アマチュア将棋指しの検討・研究
	•	自分の棋譜解析（特に勝勢でのミス分析）
	•	人間視点に最適化された対局AI
	•	将棋AIの新しい使い方・思想の検証
