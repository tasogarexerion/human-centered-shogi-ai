# Human-Centered Shogi AI

This repository contains an experimental shogi AI system
designed to make strong shogi engines usable for amateur players.

## Concept

- Does NOT aim to win AI competitions
- Focuses on human decision-making errors
- Adds a lightweight "HumanScore" layer on top of existing engines
- Automatically adapts via self-play and lightweight learning

## Target Users

- Amateur shogi players (around 1-dan level)
- Developers interested in human-centered AI design
